# BystroMarket: Анализ результатов A/B-тестирования новой платформы

Проект посвящен оценке эффективности новой платформы e-commerce сервиса «BystroMarket». В рамках работы была проанализирована воронка продаж, проверена статистическая значимость изменений конверсии пользователей в добавление товара в корзину, рассчитана величина полученного эффекта с помощью доверительных интервалов, а также разработан интерактивный дашборд для мониторинга метрик.

## Стек технологий
* **Базы данных:** PostgreSQL, SQLAlchemy (выгрузка и интеграция данных)
* **Язык программирования:** Python
* **Библиотеки:** Pandas, NumPy, Matplotlib, Seaborn, SciPy (stats)

## Окружение и исходные данные
*Проект реализован в формате сквозного кейса: для имитации реальной продуктовой среды была спроектирована локальная база данных PostgreSQL, наполненная сгенерированными синтетическими данными на основе реальных e-commerce метрик.*

## Этапы исследования

### 1. Выгрузка и предобработка данных
* Настроено подключение к базе данных PostgreSQL с использованием `SQLAlchemy`.
* Написан SQL-запрос для объединения таблиц сырых событий (`raw.events`) и профилей пользователей (`raw.users`) через `JOIN`.
* Проведен парсинг вложенных JSON-структур в поле `metadata` для извлечения стоимости товаров.
* Проверена корректность разбиения пользователей по тестовым группам:
  * **Группа A (Контроль):** 11,676 пользователей
  * **Группа B (Тест):** 11,707 пользователей
  * Распределение трафика близко к 50/50.

### 2. Исследовательский анализ данных (EDA) и Воронка продаж
* Рассчитаны базовые метрики активности: среднее и медианное количество событий на одного пользователя.
* Визуализирована динамика активности (количество событий) по дням в разрезе тестовых групп.
* Построена продуктовая воронка по ключевым этапам взаимодействия: `view` (просмотр) -> `click` (клик) -> `add_to_cart` (добавление в корзину) -> `purchase` (покупка).
* Рассчитаны конверсии как между соседними шагами, так и сквозная конверсия.

### 3. Визуализация и BI (Дашборд в DataLens)
Финальные агрегированные данные были импортированы в Yandex DataLens для создания интерактивной отчетности. На дашборде реализованы:
* Мониторинг ключевых метрик (активность, объем трафика) в динамике по дням с возможностью фильтрации по тестовым группам.
* Наглядная визуализация шагов продуктовой воронки продаж.
* Сравнительный анализ эффективности групп A и B для оперативного принятия бизнес-решений.

### 4. Статистический анализ (Проверка гипотез)
Для оценки изменения целевой метрики (**конверсии в добавление товара в корзину**) данные были агрегированы на уровне уникальных пользователей.

* **Сформулированные гипотезы:**
  * $H_0$: Конверсия в добавление в корзину между группами A и B не отличается.
  * $H_1$: Конверсия в добавление в корзину в группе B статистически значимо отличается от группы A.
* **Статистический критерий:** Для анализа таблицы сопряженности (бинарный признак: добавил товар в корзину / не добавил) применен критерий **Хи-квадрат Пирсона ($\chi^2$)**.
* **Результат теста:** Получено значение $p\text{-value} < 0.05$, что позволяет уверенно отклонить нулевую гипотезу в пользу альтернативной. Изменения статистически значимы.

### 5. Расчет величины эффекта
Чтобы оценить не только значимость, но и практическую пользу изменений для метрики `add_to_cart`, были рассчитаны следующие показатели:
* **Абсолютный прирост конверсии (Absolute Lift):** $+4.98\%$ в пользу тестовой группы B.
* **Относительный прирост (Relative Uplift):** $+19.58\%$.
* **95%-й доверительный интервал для разности долей:** составил от $+3.73\%$ до $+6.23\%$. Так как интервал не включает 0, это подтверждает стабильность и надежность полученного позитивного эффекта.

## Ограничения исследования
* В рамках данного этапа анализ проводился без учета сегментации пользователей по регионам, устройствам и источникам трафика.
* Целевой метрикой теста выступало именно промежуточное действие (добавление в корзину); влияние новой платформы на финальную конверсию в покупку и финансовые метрики (выручка, средний чек) в рамках текущего анализа не оценивалось.
