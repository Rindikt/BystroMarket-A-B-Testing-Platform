# BystroMarket A/B Testing Platform

Пет-проект, имитирующий платформу проведения A/B-тестов в интернет-магазине.

Проект генерирует пользователей и события, сохраняет их в PostgreSQL, после чего выполняется аналитическая обработка данных и строится BI-дашборд.

> Дашборд DataLens — **в разработке**

# Что реализовано

### Генерация данных

Генерируются:

- пользователи;
- разделение пользователей на группы A/B;
- регистрации пользователей;
- пользовательские сессии;
- события:
    - view;
    - click;
    - add_to_cart;
    - purchase.

Поведение пользователей строится по вероятностной воронке с разными коэффициентами для групп A и B.

Во время генерации моделируется различная активность по дням недели:

- небольшой спад в будни;
- рост активности в выходные.

---

# Аналитика

В ноутбуке `ab_test_analysis.ipynb` выполняется анализ полученных данных.

Рассматриваются:

- распределение пользователей по тестовым группам;
- количество пользователей;
- количество событий;
- ежедневная активность пользователей;
- распределение событий по дням;
- анализ продуктовой воронки;
- сравнение тестовых групп.

Для проверки различий между группами используются статистические критерии, соответствующие типу анализируемых метрик.

# Запуск

```bash
docker compose up --build
```

При первом запуске автоматически:

- создаётся база данных;
- регистрируются пользователи;
- генерируются события;
- данные становятся доступны для анализа.
