# BystroMarket: Анализ результатов A/B-тестирования новой платформы

Проект посвящен оценке эффективности новой платформы e-commerce сервиса «BystroMarket». В рамках работы была проанализирована воронка продаж, проверена статистическая значимость изменений конверсии и рассчитана величина полученного эффекта с помощью доверительных интервалов.

## Стек технологий
* **Базы данных:** PostgreSQL, SQLAlchemy (выгрузка и интеграция данных)
* **Язык программирования:** Python
* **Библиотеки:** Pandas, NumPy, Matplotlib, Seaborn, SciPy (stats)

## Этапы исследования

### 1. Выгрузка и предобработка данных
* Настроено подключение к базе данных PostgreSQL с использованием `SQLAlchemy`.
* Написан оптимизированный SQL-запрос для объединения таблиц сырых событий (`raw.events`) и профилей пользователей (`raw.users`) через `JOIN`.
* Проведен парсинг вложенных JSON-структур в поле `metadata` для извлечения стоимости товаров.
* Проверена корректность разбиения пользователей по тестовым группам:
  * **Группа A (Контроль):** 11,676 пользователей
  * **Группа B (Тест):** 11,707 пользователей
  * Распределение трафика близко к 50/50.

### 2. Исследовательский анализ данных (EDA) и Воронка продаж
* Рассчитаны базовые метрики активности: среднее и медианное количество событий на одного пользователя.
* Визуализирована динамика активности (количество событий) по дням в разрезе тестовых групп.
* Построена продуктовая воронка по ключевым этапам взаимодействия: `view` (просмотр) -> `click` (клик) -> `add_to_cart` (добавление в корзину) -> `purchase` (покупка).
* Рассчитаны конверсии как между соседними шагами (ch累積 conversion), так и сквозная конверсия из первого действия в покупку.

### 3. Статистический анализ (Проверка гипотез)
Для оценки изменения целевой метрики (конверсии в покупку) данные были агрегированы на уровне уникальных пользователей.

* **Сформулированные гипотезы:**
  * $H_0$: Конверсия в покупку между группами A и B не отличается.
  * $H_1$: Конверсия в покупку в группе B статистически значимо отличается от группы A.
* **Статистический критерий:** Для анализа таблицы сопряженности (бинарный признак: совершил покупку / не совершил покупку) применен критерий **Хи-квадрат Пирсона ($\chi^2$)**.
* **Результат теста:** Получено значение $p\text{-value} < 0.05$, что позволяет уверенно отклонить нулевую гипотезу в пользу альтернативной. Изменения статистически значимы.

### 4. Расчет величины эффекта
Чтобы оценить не только значимость, но и практическую пользу изменений, были рассчитаны следующие показатели:
* **Абсолютный прирост конверсии (Absolute Lift):** $+4.98\%$ в пользу тестовой группы B.
* **Относительный прирост (Relative Uplift):** $+19.58\%$.
* **95%-й доверительный интервал для разности долей:** составил от $+3.73\%$ до $+6.23\%$. Так как интервал не включает 0, это подтверждает стабильность и надежность полученного позитивного эффекта.

## Ограничения исследования
* В рамках данного этапа анализ проводился без учета сегментации пользователей по регионам, устройствам и источникам трафика.
* Не оценивался финансовый эффект (изменение среднего чека и совокупной выручки), исследование сфокусировано строго на конверсии в покупку.
